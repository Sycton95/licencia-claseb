# Reglas Editoriales

## Fuente y exactitud

- Toda pregunta publicable debe referenciar una fuente formal.
- El manual oficial es la fuente primaria para contenido de estudio.
- Decreto 170 y ChileAtiende sirven para reglas del examen y contexto normativo.
- Simuladores y cuestionarios públicos solo se usan como referencia de formato.

## Redacción

- El enunciado debe ser completo, inequívoco y autocontenido.
- No se aceptan stems tipo "complete la frase".
- La instrucción debe coincidir con el modo de respuesta.
- Los distractores deben ser plausibles y no absurdos.

## Publicación

- `draft`: trabajo en curso
- `reviewed`: verificado editorialmente
- `published`: visible al público
- `archived`: retirado de circulación

Ninguna pregunta debe pasar a `published` si:

- no tiene fuente
- no tiene referencia de página
- no tiene alternativa correcta válida
- no respeta el modo de respuesta
- no ha sido revisada
